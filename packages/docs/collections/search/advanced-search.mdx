# `AdvancedSearch`

L'`AdvancedSearch` est l'assemblage des 4 composants qui constituent la recherche : `Results`, `ActionBar`, `Summary` et `FacetBox`, ainsi que des actions transverses liés aus listes (mode, ajout). L'intégralité des props de ces composants se retrouve dans ses props, souvent avec le même nom ou parfois avec un nom un peu différent (exemple : `hideFacets` dans le `ListSummary`, `hideSummaryFacets` dans l'`AdvancedSearch`).
